使用帮助

快速开始

  1. 在 MiMo 开放平台 获取 API Key
  2. 选择合成模式（预置音色 / 音色设计 / 音色克隆）
  3. 输入要合成的文本
  4. 选择音色和风格
  5. 点击「开始合成」

三种合成模式

🎵 预置音色：8种精品音色开箱即用

🎨 音色设计：用文字描述自定义音色

🧬 音色克隆：上传音频样本复刻任意音色

风格控制

1. 自然语言：在「风格指令」框描述

2. 音频标签：插入 [叹气]、[笑] 等标签

3. 导演模式：角色 / 场景 / 指导三维控制
