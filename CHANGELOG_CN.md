- **Onboarding 返回按钮** — 引导支持上一步
- **焦点可见样式** — 键盘 Tab 导航时显示焦点环
- **ARIA 属性** — toggle/模态框/状态区添加无障碍属性
- **响应式布局** — 新增 600px/400px 断点，移动端适配
- **深色模式完善** — 状态提示、spinner 颜色适配

- 快捷键按钮无响应
- charcount 设置开关无效果
- setStyle 子串匹配错误（"甜"被"甜美"拦截）

- 模式切换使用脆弱的数组索引
- 模态框点击遮罩无法关闭

## [0.0.1.1] - 2026-05-28

- **可配置 API 地址** — 支持自定义 Base URL，适配不同网络环境
  - 输入框位于 API Key 上方，默认值 `https://api.xiaomimimo.com/v1`
  - 自动保存到 localStorage
  - 兼容所有 OpenAI 协议的 MiMo API 端点

### 变更

### 修复

- 修复第二次点击合成按钮显示"已取消合成"的 bug（addEventListener 与 onclick 冲突）

## [2.5.0] - 2026-05-28

### 新增

- **音色设计模式** — 用自然语言描述自定义音色（`mimo-v2.5-tts-voicedesign` 模型）
  - 维度辅助标签（性别年龄、音色质感、情绪语气、语速节奏）
  - 6 个预设音色描述模板（ASMR、深夜电台、纪录片旁白等）
  - `optimize_text_preview` 开关，自动生成适配文本
- **音色克隆模式** — 上传音频样本复刻任意音色（`mimo-v2.5-tts-voiceclone` 模型）
  - 拖拽上传音频文件
  - 音频预览播放器
  - 文件校验（最大 10MB，仅支持 MP3/WAV）
- **音频标签面板** — 点击插入精细控制标签
  - 4 类标签：语速节奏、情绪状态、语音特征、哭笑表达
  - 点击后在光标位置插入 `[标签名]`
- **导演模式** — 高级三维控制：角色 / 场景 / 指导
  - 4 个预设模板（古装言情、悬疑推理、喜剧搞笑、史诗宏大）
  - 一键合成为风格指令
- **唱歌模式** — 自动添加 `(唱歌)` 标签
- **批量合成** — 多段文本逐个合成，显示进度
- **历史搜索** — 按文本、音色、风格搜索
- **历史导出 / 导入** — JSON 格式
- **新手引导** — 首次访问 4 步分步教程
- **快捷键**
  - `Ctrl+Enter` — 开始合成
  - `Ctrl+S` — 下载音频
  - `Ctrl+Shift+S` — 保存到历史
  - `Ctrl+D` — 切换暗色模式
  - `Esc` — 关闭弹窗
  - `?` — 显示快捷键
- **文本拖拽** — 拖 `.txt` / `.md` 文件到文本框
- **请求取消** — 使用 AbortController 支持中途取消
- **快捷键弹窗** — 显示所有快捷键

### 变更

- 风格标签从 10 个扩展到 20+ 个（新增御姐音、正太音、大叔音、台湾腔、河南话、慵懒、俏皮、深沉、沙哑、甜美）
- API 地址更新为官方 `api.xiaomimimo.com`
- 历史记录上限从 50 条提升至 100 条
- 波形动画优化（正确清理动画帧）

### 修复

- 暗色模式切换 ID 大小写问题
- 合成结束后波形动画未正确停止
